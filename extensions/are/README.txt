README - ArticleRatings

== Installing ==
* Extract the files to $IP/extensions/ArticleRatings
* Add the following line to LocalSettings.php
    require_once( "$IP/extensions/ArticleRatings/ArticleRatings.php" );

== Usage ==
* See [http://www.mediawiki.org/wiki/Extension:ArticleRatings]

== About ==
This extension was developed for a specific Wikia wiki, and is not as portable as it could be. If you need to use it on a separate MediaWiki installation, please don't hesitate to contact me on my [http://community.wikia.com/wiki/Message_Wall:UltrasonicNXT Wikia talk page].
